FAQ

Частые вопросы

Расширение автоматически перенаправляет ссылки загрузки плагинов JetBrains с plugins.jetbrains.com на рабочие зеркальные серверы (downloads.marketplace.jetbrains.com, download-cdn.jetbrains.com), что позволяет обходить блокировки и ограничения доступа.
Да! Расширение имеет открытый исходный код и не отправляет никакие данные на сторонние серверы. Все редиректы выполняются только на официальные зеркальные серверы JetBrains. Вы можете проверить код в GitHub репозитории.
Расширение поддерживает автоматическое обновление. При выпуске новой версии Chrome автоматически загрузит и установит обновление. Также вы можете вручную скачать новую версию из раздела Releases на GitHub.
Да, в настройках расширения есть whitelist — список доменов, для которых редирект не будет применяться. Также можно временно отключить расширение через popup панель.
Откройте popup панель расширения (кликните по иконке в панели браузера) или перейдите на страницу настроек через chrome://extensions/. Там вы увидите общую статистику, статистику по правилам и историю редиректов.
